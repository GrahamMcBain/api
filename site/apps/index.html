Moov Apps - API Documentation

  • Accounts - General Ledger and financial account service with an HTTP API
  • ACH - Automated Clearing House library implementing NACHA file creation and validation
  • Customers - Registry supporting Know Your Customer (KYC), Customer Identification Program (CIP), and OFAC checks
  • Fed - ABA Routing Number and Bank Name Lookup
  • Image Cash Letter (ICL) - X9's Specifications for ICL (Image Cash Letter) to provide Check 21 services
  • Paygate - RESTful API enabling electronic payments to be submitted and received without a deep understanding payment file specification
  • Watchman - AML/CTF/KYC/OFAC Search of global watchlist, sanctions, and politically exposed person (PEP)
  • Wire - FedWire funds service file parser and writer
